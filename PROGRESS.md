| เฟส | ความคืบหน้า |
|-----|-------------|
| P0 วางรากฐาน | ✅ 100% |
| P1 Core Loop | ✅ 100% |
| P2 Content & Cards | ✅ ~95% (เหลือขยายคลังภารกิจ/โชค) |
| P3 Dual Layout | ✅ ~90% (เหลือ a11y เต็ม + TTS) |

**เล่นได้แล้ว:** โหมดครูตั้งค่า → เริ่มเกม → ทอยเต๋า (มีเสียง+สั่น) → เดินหมาก → ตอบคำถาม (จับเวลา, คัดตามความยาก) / ภารกิจ mini-game (เรียงลำดับ·จับคู่) / รับเหรียญ / ปลดล็อกมหาราช / การ์ดโชค → เปิด AR (เดโม) → สลับแนวตั้ง/แนวนอน → **จบเกม → สรุปอันดับ + badge → เล่นอีกครั้ง**

## 🎯 คิวถัดไป (แนะนำ)
1. **AR จริง** — เชื่อม MindAR/AR.js + โมเดล 3D 7 พระองค์ (.glb/.usdz) + AR marker (เฟส P4) *ต้องมีไฟล์โมเดล 3D*
2. **Accessibility** — TTS อ่านออกเสียงคำถาม/ความรู้, ตรวจคอนทราสต์ WCAG AA
3. **PWA** — ติดตั้งลงแท็บเล็ต + เล่นออฟไลน์ (manifest + service worker)
4. **อนิเมชันฉลอง** ตอนตอบถูก/ปลดล็อก + ตรวจเนื้อหากับครู
